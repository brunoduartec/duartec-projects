O sistema hidráulico em questão é uma simplificação do sistema de uma aeronave comercial regional. Esta aeronave possui três sistemas hidráulicos independentes, responsáveis por fornecer potência hidráulica para as diversas cargas, tais como superfícies de controle primárias e spoilers, trens de pouso, entre outras. Algumas cargas, como as superfícies de controle primárias, podem ser acionadas a partir de sistemas independentes. A potência hidráulica a ser distribuída é gerada por bombas, acionadas de diferentes formas. Duas bombas são acionadas cada uma por um motor da aeronave. Quatro bombas são acionadas por motores elétricos de corrente alternada, cuja alimentação provém de diferentes fontes.
