\section{Definição de parâmetros}

Os testes serão realizados adicionando variáveis de forma artificial por meio de transformações afins de forma a emular algum comportamento ou situação encontrada no ambiente de manutenção
\begin{itemize}
	\item Variação de escala para simular a aproximação dentro da janela de
	inspeção
	\item Variação de rotação para simular a movimentação durante a manutenção
	\item Variação de iluminação para simular manutenções feitas em horários do dia
	e iluminação diferente como por exemplo ambientes com neve com brilho muito maior
	\item Adição de Blur para emular ambientes com muita poeira ou esfumaçados
\end{itemize}

A definição das faixas de parâmetro utilizados foi feita com inspeção em campo

\section{Análise de dados}

A análise de desempenho de cada técnica deve ser balizada por parâmetros que garantam que a aplicação seja viável para o contexto citado na seção\ref{sec:contexto}. Para tal deve-se considerar

\begin{itemize}
	\item Cadência > 5fps ou seja, tempo < 200ms [REFERENCIA]
	\item Robustez à escala > X1
	\item Robustez à rotação >X2
	\item Robustez à iluminação >X3
	\item Robustez à blur >X4
\end{itemize}
