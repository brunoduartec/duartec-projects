\section{Objetivos}
O presente trabalho tem como objetivo geral avaliar o ambiente de manutenção
aeronáutico, no contexto de janelas de inspeção, traçando estratégias de
reconhecimento de items de manutenção.
Para a consecução do objetivo geral, foram definidos os seguintes objetivos específicos:


\begin{itemize}
\item Avaliar os algoritmos clássicos de reconhecimento;
\item Aplicar os algoritmos clássicos à situações reais;
\item Selecionar algoritmo mais adequado para o contexto.
\end{itemize}
	
	
Esta tese propõe o uso da realidade aumentada no cenário de manutenção de
aeronaves, tendo como objetivo determinar a melhor estratégia de reconhecimento
de características dos objetos no contexto, para que posteriormente seja
utilizado em ferramentas de auxilio na manutenção.
