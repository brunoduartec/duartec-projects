O reconhecimento de objetos em uma cena para posterior uso em realidade aumentada 
depende de diversas variáveis, causando a necessidade do uso de diversas técnicas 
específicas para cada cenário, sendo portanto, um estudo de fronteiras para a melhor escolha 
do algoritmo de registro de acordo com a aplicação em questão de grande valia para o meio 
acadêmico. 
Esta tese se propõe a estudar, categorizar e traçar fronteiras das técnicas conhecidas tendo 
como caso de uso manutenção de aeronaves feita dentro de centros fechados.
