Para compreender o presente trabalho se faz necessário elucidar fundamentos
básicos, fornecendo assim subsídios necessários para o leitor avaliar com mais
propriedade os resultados apresentados. Os conceitos apresentados seguem uma
ordem crescente de conhecimento para que seja construído um arcabolso de
conhecimento suficiente.
O capítulo inicia com uma explanação sobre realidade virtual e realidade
aumentada, elucida o uso de dispositivos para imersão. Para a realização de
aplicações de realidade aumentada o uso de sensores para abstrair informações do
ambiente é fundamental, sendo as câmeras de baixo custo os mais comuns,
portanto, é descrito no capítulo os problemas de distorção inerentes a tais
dispositivos e a modelagem matemática adotada para as câmeras utilizada em todas
as abordagens presentes nesse trabalho.
A abordagem utilizada nesse trabalho é de reconhecimento de características
locais, diferente do conceito comum de reconhecer padrões como retângulos,
circulos, ou contornos, portanto conceituar o reconhecimento de características
se faz tão importante.
É também apresentada conceituação básica de cada um dos algoritmos utilizados e
de parâmetros utilizados para a análise dos resultados
